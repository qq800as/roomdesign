# Changelog

## 0.1.0 - 2026-06-16

- Initial release of AutoCAD Interior MCP.
- Added AutoCAD COM client connection and document operations.
- Added interior drafting tools:
  - layer setup
  - wall and room drawing
  - door/window symbols
  - furniture placement (proxy/block)
  - text labels
  - aligned dimensions
  - hatch by boundary handle
  - raw command passthrough
- Added installation, usage, and publishing documentation.
